Este diretório contém a documentação técnica oficial do projeto **MinhaTirinha**.

O MinhaTirinha é um aplicativo focado no alívio do estresse através da arteterapia digital, permitindo que os usuários pintem histórias em quadrinhos em tons pastéis com música de fundo relaxante.
